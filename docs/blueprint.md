# **App Name**: Portal do Aluno EBD

## Core Features:

- Student Login: Secure login for students to access their assigned activities.
- Activity Dashboard: Display of weekly Bible reading assignments (2 chapters per reading, with multiple choice questions).
- Bible Project Integration: Display of weekly Bible Project video assignments with links to YouTube and short answer questions.
- Module Management: Functionality for teachers to assign students to modules.
- Assignment Correction: Interface for teachers to review and grade student answers.
- Progress Reporting: Reporting dashboard for teachers to review student performance data and for parents to check their child's progress.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and knowledge.
- Background color: Light gray (#F5F5F5) for a clean, modern look.
- Accent color: Gold (#FFC107) to highlight important information and calls to action.
- Body text: 'PT Sans', a humanist sans-serif, suitable for body text.
- Headline text: 'Alegreya', a humanist serif font, elegant, intellectual, suitable for headlines
- Use simple, intuitive icons to represent modules and activities.
- A clean, well-organized layout with clear section headings and consistent design elements.
